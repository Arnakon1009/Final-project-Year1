# Final-project-Year1
Final project for Programming 2 | Computer Engineering, Year 1 at KOSEN-KMITL (AY2025)

This application was co-developed with my teammate as our final submission for the Programming 2 course.

**Prerequisites**
  Before using this application, please obtain an API key from: ExchangeRate-API (https://www.exchangerate-api.com/)

**Installation Guide (Step-by-Step)**
  Download and Install: Download the .zip file and ensure Python is installed (Recommended: Python 3.10.x).

  Extract: Extract the contents of the .zip file.

  Configure API Key: Open the .env.example file and replace the placeholder with your personal API key.

  Rename Environment File: Rename .env.example to .env.

  Once configured, you can launch the application by running run_this_file.pyw.

**Project Purpose**
  This application was developed as a case study to practice and implement the Tkinter module.
